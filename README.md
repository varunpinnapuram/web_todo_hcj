# Todo App — HTML/CSS/JavaScript (Portfolio Project)

## Summary
A clean, single-page Todo application demonstrating:
- Semantic HTML structure
- Responsive CSS layout
- Vanilla JavaScript (DOM manipulation and localStorage)

## Files
- `index.html`
- `styles.css`
- `script.js`
- `screenshot.png` — simple preview (auto-generated)
- `README.md`

## How to run locally
1. Open `index.html` in a modern browser.
2. Add, complete, delete tasks. Tasks are persisted via `localStorage`.
3. To host on GitHub Pages: push this repo and enable Pages from the `main` branch.
